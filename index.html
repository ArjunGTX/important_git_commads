GIT COMMANDS

the most commonly used git commands include :

git config --global user.name "user name" sets the user name for git.

git config --global user.email "email address" sets the user email address for git.

git init Initializes git in the current folder.

git add "fileName" Adds the file to the staging area.

git add * Adds all the files in the current folder.

git commit -m "message" commits the staged files into the local git repository.

git status Shows whether the files are staged or committed.

git log displays all the commits in order with the corresponding commit Id's.

git remote add remoteName "remote repository link" links the local repository to the remote repository.

git push -u remoteName branchName uploads the files in local repository to the remote repository.

git clone /path/to/repository creates a copy of local repository.

git clone username@host:/path/to/repository creates a copy of remote repository.

git remote -v List all currently configured remote repositories.

git checkout -b branchName Create a new branch and switch to it.

git checkout branchName Switch from one branch to another.

git checkout branchName Switch from one branch to another.

git branch> List all the branches in your repo, and also tell you what branch you're currently in.

git push --all origin Push all branches to your remote repository.

git branch -d branchName Deletes the branch.

git pull Fetch and merge changes on the remote server to the working directory.

git merge branchName merges a different branch into the active branch.
